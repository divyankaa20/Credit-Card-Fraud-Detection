# 💳 Credit Card Fraud Detection

A machine learning project that detects fraudulent credit card transactions by analyzing patterns and anomalies in financial data.

---

## 🚀 Features
- Detects fraudulent transactions using ML algorithms  
- Analyzes transaction patterns  
- Helps in reducing financial risk  
- Improves fraud detection accuracy  

---

## 🛠️ Technologies Used
- Python 🐍  
- Machine Learning 🤖  
- Pandas, NumPy  
- Scikit-learn  

---

## 📂 Project Structure
- `main_train.py` → Model training file  
- `app/` → Application files  
- `models/` → Saved model files  
- `requirements.txt` → Dependencies
